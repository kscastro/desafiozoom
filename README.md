# Desafio Zoom

## Ambiente

  Para você conseguir rodar este projeto, tem que instalar os componentes listados tanto no linux quanto no mac.

**NODE 12**
```bash
  sudo apt install nodejs
```
**NPM**
```bash
  sudo apt install npm
```

## Rodando o projeto

```javascript
  npm install
```
```javascript
  npm start
```
